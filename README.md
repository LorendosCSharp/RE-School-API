# RE-School-API
## API to communicate with XSCHOOL trough simple node-fetch 

## Functionality
- Login with / without 2FA
- Code verification
- Getting information about the given user

## Build
- clone the repo
- navigate to the root of the repo on your machine
- type ``npm build``
- use the dist folder in your code
